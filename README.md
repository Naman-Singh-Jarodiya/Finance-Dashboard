# 💰 FinTrack Pro Dashboard

A simple web app to track income and expenses with CSV and PDF export.

## 🚀 Features

* Add & manage transactions
* View balance, income, expenses
* Download CSV data
* Generate PDF report
* Light/Dark mode

## 🛠️ Tech

* HTML
* Tailwind CSS
* JavaScript

## ▶️ Run

Open `index.html` in your browser.
